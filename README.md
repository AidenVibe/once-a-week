# 주에한번은

부모님께 보낼 질문을 매일 제공하는 서비스.

## 사용하기

- **웹**: https://aidenvibe.github.io/once-a-week/
- **텔레그램 봇**: [@mwohae_bot](https://t.me/mwohae_bot)

## 구조

```
web/          # 웹사이트 (GitHub Pages)
bot/          # 텔레그램 봇 (Python)
```

## 로컬 실행

```bash
# 봇 실행
cd bot
pip install -r requirements.txt
# .env 파일에 BOT_TOKEN 설정 필요
python main.py
```
